# Haste
![banner](https://i.ibb.co/mhMMQrQ/hastebanner.png)
Haste is a simple typing speed practice/tester made with Svelte!

Haste gathers text through the quotable API, which contains thousands of clean and interesting quotes to type out. If you're interested, you can check it out (by Luke Peavy!)
